    This is an experimental technology
    Because this technology's specification has not stabilized, check the compatibility table for usage in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the specification changes.

     

    The getAnimations() method of the Document interface returns an array of all Animation objects currently in effect whose target elements are descendants of the document.

    Syntax

    var allAnimations = document.getAnimations();

    Parameters

    None.

    Return value

    Void.

    Examples

    The following code snippet will slow down all animations on a page by halving their Animation.playbackRate.

    document.getAnimations().forEach(
  function (animation) {
    animation.playbackRate *= .5;
  }
);
