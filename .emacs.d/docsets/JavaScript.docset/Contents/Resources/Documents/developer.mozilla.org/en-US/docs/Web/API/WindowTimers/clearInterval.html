    WindowTimers.clearInterval()

    In This Article
    1. Syntax
    2. Example
    3. Specification
    4. See also

    Cancels repeated action which was set up using setInterval.

    Syntax

    window.clearInterval(intervalID)

    intervalID is the identifier of the repeated action you want to cancel. This ID is returned from setInterval().

    Example

    See the setInterval() example.

    Specification

    Specification Status Comment
    WHATWG HTML Living Standard
    The definition of 'clearInterval' in that specification.    		 Living Standard
