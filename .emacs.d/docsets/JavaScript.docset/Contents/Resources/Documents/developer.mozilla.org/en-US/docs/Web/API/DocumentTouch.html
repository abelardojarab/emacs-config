    DocumentTouch

    by 5 contributors:
    Show Sidebar
    In This Article
    1. Methods
    2. See also

    Obsolete since Gecko 25 (Firefox 25 / Thunderbird 25 / SeaMonkey 2.22)
    This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

    The DocumentTouch interface used to provides convenience methods for creating Touch and TouchList objects, but has been removed. The methods live on the Document interface now.

    Methods

    DocumentTouch.createTouch()
    Creates a new Touch object.
    DocumentTouch.createTouchList()
    Creates a new TouchList object.
