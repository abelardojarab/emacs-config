    ImageData.height

    In This Article
    1. Syntax
    2. Example
    3. Specification
    4. Browser compatibility
    5. See also

    The readonly ImageData.height property returns the number of rows in the image data object.

    Syntax

    imagedata.height

    Example

    var imagedata = new ImageData(100, 100);
imagedata.height; // 100
