    WebGL2RenderingContext.isSync()

    In This Article
    1. Syntax
      1. Parameters
      2. Return value
    2. Examples
    3. Specifications
    4. Browser compatibility
    5. See also

    This is an experimental technology
    Because this technology's specification has not stabilized, check the compatibility table for usage in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the specification changes.

    The WebGL2RenderingContext.isSync() method of the WebGL 2 API returns true if the passed object is a valid WebGLSync object.

    Syntax

    GLboolean gl.isSync(sync);

    Parameters

    sync
    A WebGLSync object to test.

    Return value

    A GLboolean indicating whether the given object is a valid WebGLSync object (true) or not (false).

    Examples

    gl must be a WebGL2RenderingContext. WebGLSync objects are not available in WebGL 1.

    var sync = gl.fenceSync(gl.SYNC_GPU_COMMANDS_COMPLETE, 0);

// ...

gl.isSync(sync);
