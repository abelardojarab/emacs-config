    HTMLTableElement.border

    In This Article
    1. Syntax
    2. Example
    3. Notes
    4. Specification

    Deprecated
    This feature has been removed from the Web standards. Though some browsers may still support it, it is in the process of being dropped. Do not use it in old or new projects. Pages or Web apps using it may break at any time.

    The HTMLTableElement.border property represents the border width of the table element.

    Syntax

    HTMLTableElement.border =
border;
var
border =
HTMLTableElement.border;
    • border is string representing the width of the border in pixels.

    Example

    // Set the width of a table border to 2 pixels
var t = document.getElementById("TableA");
t.border="2";

    Notes

    This attribute is deprecated in HTML 4.0.

    Specification

    • W3C DOM 2 HTML Specification HTMLTableElement .border.
