    BluetoothAdapter.setPasskey()

    by 4 contributors:

    Non-standard
    This feature is not on a current W3C standards track, but it is supported on the Firefox OS platform. Although implementations may change in the future and it is not supported widely across browsers, it is suitable for use in code dedicated to Firefox OS apps.

    This API is available on Firefox OS for internal applications only.

    Summary

    The setPasskey method is used to send back the requested Passkey code when the device's adapter tries to pair itself with a remote device.

    Syntax

    var request = instanceOfBluetoothAdapter.setPasskey(deviceAddress, key);

    Parameters

    deviceAddress
    The Bluetooth micro-network address of the device for which the confirmation is required.
    key
    A number representing the Passkey code set by the user.

    Returns

    A DOMRequest object to handle the success or error of the operation.

    Specification

    Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
