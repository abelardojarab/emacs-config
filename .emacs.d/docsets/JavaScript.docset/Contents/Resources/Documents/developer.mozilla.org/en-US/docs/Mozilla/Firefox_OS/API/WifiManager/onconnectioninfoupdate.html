    WifiManager.onconnectioninfoupdate

    In This Article
    1. Summary
    2. Syntax
    3. Specifications
    4. See also

    This API is available on Firefox OS for internal applications only.

    Summary

    Specifies an event listener to receive connectioninfoupdate events. Those events occur when the device's WiFi connection information changes.

    Syntax

    navigator.mozWifiManager.onconnectioninfoupdate = funcRef

    Where funcRef is a function to be called when the connectioninfoupdate event occurs. This callback function receives a MozWifiConnectionInfoEvent object as the first parameter.

    Specifications

    Not part of any specification.
