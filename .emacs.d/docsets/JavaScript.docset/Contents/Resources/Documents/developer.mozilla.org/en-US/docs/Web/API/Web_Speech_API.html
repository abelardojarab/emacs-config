    This is an experimental technology
    Because this technology's specification has not stabilized, check the compatibility table for the proper prefixes to use in various browsers. Also note that the syntax and behavior of an experimental technology is subject to change in future versions of browsers as the spec changes.

    The Web Speech API allows incorporatation of voice data into a web app. There are two parts to the Web Speech API : SpeechSynthesis (Text-to-Speech) and SpeechRecognition (Asynchronous Speech Recognition.)

    Web Speech concepts and usage

    The Web Speech API makes web apps able to handle voice data. There are two components in this API. SpeechSynthesis is a text-to-speech component, allowing programs to read their text content. The other is SpeechRecognition, which provides the capability of recognizing voice context from an audio input and responding appropriately.

    Note: You can read more about using the Web Speech API in our article Using the Web Speech API (TBD - write basic tutorial).

    Web Speech API Interfaces

     
    SpeechSynthesis
    The controller interface for the speech service. The speech service can be paused, resumed, removed or assigned a task through this interface.
    SpeechSynthesisUtterance
    A SpeechSynthesisUtterance indicates a speech request. It contains the content the speech service should read and information about how to read it (e.g., language, pitch and volume.)
    SpeechSynthesisVoice
    Represents a voice that the system supports. Every SpeechSynthesisVoice has its own relative speech service including the information about language, name and URI.
    SpeechSynthesisVoiceList
    A list of voices (SpeechSynthesisVoice objects) that the system supports.
    SpeechSynthesisEvent
    Contains information about the current state of the SpeechSynthesisUtterance that has been processed in the speech service.
    SpeechRecognition
    The controller interface for the recognition service; this also handles the SpeechRecognitionEvent sent from the recognition service.
    SpeechRecognitionAlternative
    Represents the raw word that user speaks.
    SpeechRecognitionError
    Represents error messages from the recognition service.
    SpeechRecognitionEvent
    Contains the interim or final resilt.
    SpeechRecognitionResult
    The n-best of the SpeechRecognitionAlternative.
    SpeechRecognitionResultList
    Represents a list of SpeechRecognitionResult objects, or a single one if in continuous mode.
    SpeechGrammarList
    Represents a list of SpeechGrammar objects.
    SpeechGrammar
    The words or patterns of words that we want the recognition service to recognize.
