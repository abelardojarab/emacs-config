    Coordinates.longitude

    In This Article
    1. Syntax
    2. Specifications
    3. Browser compatibility
    4. See also

    The Coordinates.longitude read-only property is a double representing the longitude of the position in decimal degrees.

    Syntax

    lon = coordinates.longitude
