    Navigator.mozL10n

    In This Article
    1. Syntax
    2. Value
    3. Specifications
    4. See also

    Non-standard
    This feature is not on a current W3C standards track, but it is supported on the Firefox OS platform. Although implementations may change in the future and it is not supported widely across browsers, it is suitable for use in code dedicated to Firefox OS apps.

    navigator.mozL10n returns an L10n object you can use to access localization resources and respond to localization events.

    Syntax

    var l10n = navigator.mozL10n;

    Value

    navigator.mozL10n returns an L10n object.

    Specifications

    Not part of any specification; however, note that this API will be replaced by L20n in the future.
