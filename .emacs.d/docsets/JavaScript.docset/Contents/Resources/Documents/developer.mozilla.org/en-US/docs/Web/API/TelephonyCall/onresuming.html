    TelephonyCall.onresuming

    Non-standard
    This feature is not on a current W3C standards track, but it is supported on the Firefox OS platform. Although implementations may change in the future and it is not supported widely across browsers, it is suitable for use in code dedicated to Firefox OS apps.

    This API is available on Firefox OS for internal applications only.

    The onconnected event handler of the TelephonyCall interface is a reference to a function to call when a previously held call resumes.

    The event handler is called with a CallEvent as its only parameter.

    Syntax

    myCall.onresuming = function() { ... };

    Examples

    // Telephony object
var tel = navigator.mozTelephony;

// Place a call
tel.dial("123456789").then(function(call) {
  call.number;

  call.onresuming = function(e) {
    console.log('Your call to ' + e.call.id + ' is resuming.');
  }
});
