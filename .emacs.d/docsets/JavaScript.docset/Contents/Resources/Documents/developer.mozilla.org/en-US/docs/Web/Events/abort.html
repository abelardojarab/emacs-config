    abort

    by 5 contributors:
    Show Sidebar
    In This Article
    1. General info
    2. Properties

    The abort event is fired when the loading of a resource has been aborted.

    General info

    Specification
    DOM L3
    Interface
    UIEvent if generated from a user interface, Event otherwise.
    Bubbles
    No
    Cancelable
    No
    Target
    Element
    Default Action
    None

    Properties

    Property Type Description
    target Read only EventTarget The event target (the topmost target in the DOM tree).
    type Read only DOMString The type of event.
    bubbles Read only boolean Does the event normally bubble?
    cancelable Read only boolean Is it possible to cancel the event?
    view Read only WindowProxy document.defaultView (window of the document)
    detail Read only long (float) 0.
