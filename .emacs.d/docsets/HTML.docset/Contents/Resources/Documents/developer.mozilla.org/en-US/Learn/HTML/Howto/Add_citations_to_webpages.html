    Learn how to quote someone's exact words, and give your source for them.

    Prerequisites: You need to be familiar with how to create a basic HTML document.
    Objective: Learn how to mark direct quotations and their source.

    We're always talking about what other people said, and building arguments on it, especially in technical literature, journalism, or philosophy. Of course, since scientists designed HTML in the first place, they built several quoting mechanisms into HTML.

    HTML accounts for the semantics of three quotation-related use cases:

    • providing a full excerpt from a third-party document
    • quoting a single phrase or word
    • referencing to a third-party document

    How to report someone's exact words

    Inline quotation

    HTML provides the <q> element for inline quotations, that is, quotes short enough to embed right in your text.

    <p>
  If you read Cicero in Latin, you might find a mysteriously
  familiar passage: <q lang="la">qui dolorem ipsum quia dolor
  sit amet consectetur adipisci velit.</q>
</p>

    Here's how it looks:

    Do not use <q> for chapter/article titles or as scare quotes. If you must use quotation marks for these purposes, you'll need to handwrite quote marks, like this:

    Don’t say “utilize” when you mean “use”.

    Feel free to use the corresponding HTML character entities instead: &ldquo; and &rdquo; for double quotes, &lsquo; and &rsquo; for single quotes.

    By default, Web browsers surround the quotation with the quote marks appropriate for the language in use. If necessary, you can override the default styling with the CSS quotes property. The following CSS rule imposes the quotation marks appropriate for French:

    :lang(fr) > q {
  quotes: '« ' ' »';
}

    Full excerpt

    HTML also furnishes <blockquote> for block quotations (quotes long enough to merit their own paragraph(s)). The following example shows how you can continue to declare paragraphs inside a <blockquote>:

    <p>Cicero's words have become garbled for use as dummy 
text, but originally contained sound advice:</p>

<blockquote>
  <p>
    Nor again is there anyone who loves or pursues or 
    desires to obtain pain of itself, because it is pain,
    but occasionally circumstances occur in which toil
    and pain can procure him some great pleasure. . . .
  </p>

  <p>
    The wise man therefore always holds in these matters
    to this principle of selection: he rejects pleasures
    to secure other greater pleasures, or else he endures
    pains to avoid worse pains.
  </p>
</blockquote>

    In this case the default styling used by web browser is a wider left margin. Again, you're free to override the default styling by using the CSS margin property.

    How to provide a source for a quote

    Implicit sourcing

    Quoting is good, but citing is better. An independent source can establish that you're quoting correctly, and also provide the broader context for the quote. Theoretically, there's a nice cite attribute, which you can use to associate your source's URL with either <q> or <blockquote>:

    <q cite="https://en.wikiquote.org/wiki/Homer">The gods know all things.</q>

    In practice, people will never find that information unless they happen to be reading your HTML source. It's probably better to just use a regular link with the quote:

    <q><a href="https://en.wikiquote.org/wiki/Homer">The gods know all things.</a></q>

    Explicit sourcing

    To explicitly tell people where you got the quote, you can use the <cite> element. This element can either indicate the quote's author or the creative work the quote comes from.

    <p>
  <q>Everything has a small beginning</q>,
  <cite id="author">Cicero</cite> wrote in
  <cite id="work" lang="la">De Finibus Bonorum et Malorum</cite>.
</p>

    To make the relationship between the quote and its sources extra explicit, you can add the aria-labelledby attribute:

    <p>
  <q aria-labelledby="author work">Everything has a small beginning</q>,
  <cite id="author">Cicero</cite> wrote in
  <cite id="work" lang="la">De Finibus Bonorum et Malorum</cite>.
</p>
