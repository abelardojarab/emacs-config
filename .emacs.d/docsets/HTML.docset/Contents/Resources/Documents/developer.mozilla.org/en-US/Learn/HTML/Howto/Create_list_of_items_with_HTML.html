    Create list of items with HTML

    by 4 contributors:
    Show Sidebar
    In This Article

    2. Building lists with HTML

      2. How to change the numbers/bullets

    Need an automatically numbered list? Need bullet points? Whatever kind of list you need, we'll get it set up in a flash.

    Prerequisites: You need to be familiar with how to create a basic HTML document.
    Objective: Set up an ordered or unordered list with HTML.

    Unordered and ordered lists

    Unordered lists

    As the name implies, unordered lists don't lose meaning when you shuffle the items. Normally each list item has a bullet to keep it distinct. A shopping list is a great example, since it doesn't matter in what order you buy your things:

    Unordered lists tend to crop up in places you weren't expecting—navigation bars, for instance, are just unordered lists of links with fancy CSS styling.

    Ordered lists

    Here, the items must be read or done in some order, so the items are usually numbered. Step-by-step processes are a great illustration, since you can't do any step until you've done the previous steps:

    Building lists with HTML

    So, how do you make an HTML list? First, answer just one question: if you shuffle the list items, does the meaning change?

    • If so, use the <ol> element to create an ordered list.
    • If not, use the <ul> element to create an unordered list.

    Add list items

    Your entire list goes inside <ol> or <ul> tags. Each list item goes inside <li> tags. Here's an example of both kinds of lists:

    <h1>Quick Guacamole</h1>

<h2>Ingredients</h2>
<ul>
  <li>2 avocados with pit and peel removed</li>
  <li>juice from 1 lime</li>
  <li>¼ cup chopped cucumber</li>
  <li>1 small tomato, chopped</li>
</ul>

<h2>Instructions</h2>
<ol>
  <li>Gently break up the avocado halves with a fork</li>
  <li>Place in a dish and squirt the lime juice</li>
  <li>Mix so the lime juice keeps the avocado from getting brown</li>
  <li>Stir in the chopped cucumber and tomato</li>
  <li>Keep refrigerated, and serve promptly with tortilla chips and (if possible) fresh cilantro</li>
</ol>

    By default, your browser numbers list items whose order matters, but displays bullets otherwise:

    How to change the numbers/bullets

    with HTML

    • To start with some number besides 1, use the start attribute on the <ol> tag: <ol start="3">.
    • To count down instead of up (e.g., 5-4-3-2-1), use the reversed attribute on the <ol> tag: <ol reversed>.
    • To change from numbers to letters, use the type attribute on the <ol> tag: <ol type="a"> (type="1" for number, ="a" for lowercase letter, ="A" for uppercase letter, ="i" for lowercase Roman numeral, ="I" for uppercase Roman numeral).
    • To change a specific list item's number, use value attribute on the <li> tag: <li value="5">. (All subsequent list items are also renumbered.)

    For example, the following ordered list uses uppercase letters starting from "C":

    <ol start="3" type="A">
  <li>I'm first</li>
  <li>I'm second</li>
  <li>I'm third</li>
</ol>

    It looks like this:

    or a touch of CSS

    HTML attributes are for changing the content (e.g., if you want your list to start at step 5 rather than step 1). For simply cosmetic adjustments, you'll want CSS.

    • To change numbering style or bullet style, use CSS list-style-type
    • To use a custom image as bullet, use CSS list-style-image. Do this with care; images are slower, heavier, and less scalable than text.

    CSS can help you achieve some very complex effects, like custom counting systems. If you're looking to get into CSS, take a look at our CSS Guide and check how to make your CSS rules take effect in your webpage.

    Quick example. Here's an HTML unordered list:

    <ul>
  <li>I got a dot</li>
  <li>I got a circle</li>
  <li>I got a circle too</li>
</ul>

    Per these CSS rules, all list items except the first receive a hollow circle as a bullet:

    ul {
  list-style-type: circle;
}

li:first-child {
  list-style-type: disc;
}

    Nesting lists

    You'll often see lists nested freely inside each other. In other words, you can put a whole list inside a list item:

    <ul>
	<li>a component</li>
	<li>a process:
		<ol>
			<li>step 1</li>
			<li>step 2</li>
			<li>step 3</li>
		</ol>
	</li>
	<li>a component</li>
</ul>
