    Annotate images and graphics

    by 3 contributors:
    Show Sidebar
    In This Article
    1. What is a figure?
    2. Add a caption
    3. Learn more

    HTML provides a simple way to mark up figures and their captions.

    Prerequisites: You need to be familiar with how to create a basic HTML document.
    Objective: Learn how to add figures to your webpage, complete with captions.

    What is a figure?

    A figure is an independent unit of content that:

    • expresses your meaning in a compact, easy-to-grasp way
    • could go in several places in the page's linear flow
    • provides essential information supporting the main text

    When you find yourself wanting to add a caption, you probably have a figure on your hands. Here are some examples: images, code snippets, audio and video, equations, tables, diagrams.

    To mark up figures, HTML provides the convenient <figure> element:

    <figure>
  <img alt="MDN" src="https://mdn.mozillademos.org/files/6457/mdn_logo_only_color.png">
</figure>

    Add a caption

    Enclose your caption in <figcaption>, and place the <figcaption> element within the <figure> element, next to either <figure> tag:

    <figure>
  <img alt="" src="https://mdn.mozillademos.org/files/6457/mdn_logo_only_color.png">
  <figcaption>The regular MDN logo in 2014</figcaption>
</figure>

    <figcaption> tells browsers and assistive technology that the caption describes the other contents of <figure>.

    From an accessibility viewpoint, captions and alt text have distinct roles. Captions benefit even people who can see the image, whereas alt text provides the same functionality as an absent image.

    Therefore, captions and alt text shouldn't just say the same thing, because they both appear when the image is gone. Try turning images off in your browser and see how things look.
