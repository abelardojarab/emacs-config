scipy.special.zetac¶

scipy.special.zetac(x) = <ufunc 'zetac'>¶

Riemann zeta function minus 1.

This function is defined as

\[\zeta(x) = \sum_{k=2}^{\infty} 1 / k^x,\]

where x > 1.
