#+BEGIN_EXAMPLE
#+ATTR_HTML: :class float-right 
#+ATTR_HTML: :class z-index-999

 ============merge============>
#+ATTR_HTML: :class float-right z-index-999
#+END_EXAMPLE
Otherwise, only one class will be applied in browser!
